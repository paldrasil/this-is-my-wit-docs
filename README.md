# This Is My Wit

`This Is My Wit` is a top-down 3D stealth-puzzle game concept built in Unity.
You play as a buffalo herder who cannot fight the tiger directly, so you must use timing, distraction, and positioning to guide the buffalo to safety.

## Why This Exists

This project exists to prototype and validate a gameplay-first loop (`Observe -> Distract -> Lead -> Escape`) before scaling content.
It also defines a disciplined docs workflow (`PLAN -> milestone specs -> implementation`) so design, tech, and production decisions stay aligned as the game evolves.
This repository is intentionally docs-first and docs-only: specs are versioned, cross-linked, and reviewed with lightweight guardrails to prevent non-documentation drift.
